 Dice Simulator in C

This project simulates dice rolls using the C language.

Features
- User-defined number of rolls
- Random number generation
- Frequency count
- Percentage calculation
- Text-based histogram

 Example Output
Face    Frequency    Percentage    Histogram
1       1002         16.70%        ********
2       980          16.33%        ********


- C Language
- Standard Libraries (stdio, stdlib, time)


gcc dice.c -o dice
./dice
